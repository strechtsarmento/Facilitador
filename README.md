# Facilitador Pro

Aplicativo web em HTML, CSS e JavaScript para acompanhar evolução corporal e criar comparações de fotos.

## Fluxo de foto fantasma

1. Abra **Antes e Depois**.
2. Escolha uma foto anterior ou tire uma nova.
3. Confirme a foto.
4. Toque em **Abrir Visor Alinhado**.
5. A foto anterior aparecerá translúcida sobre a câmera.
6. Ajuste a transparência, arraste para posicionar e use dois dedos ou os botões para alterar o tamanho.
7. Tire a foto atual. A imagem salva contém somente a foto nova; a referência translúcida serve apenas como guia.

## Câmera no navegador

Por segurança, navegadores normalmente liberam a câmera em `https://` ou em `localhost`. Para testar no computador, abra a pasta por um servidor local, por exemplo com a extensão Live Server do VS Code.
